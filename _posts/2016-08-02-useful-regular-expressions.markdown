Just saving some RegEx that I use :D

### Season and Episode:
```bash
[Ss]\d{2}[Ee]\d{2}
```
Example:

- S01E10
- s11E12
- S30e01

### Video Types
```bash
\.(mkv|mp4|avi|mov)
```
Example:

- mkv
- mp4
- avi
